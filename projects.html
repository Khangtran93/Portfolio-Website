Snake Game

This classic snake game was created in Python using the Turtle module. The player controls a snake that moves around the screen and grows as it eats food. Flash Card App

This handy Flash Card app developed in Python using tkinter module helps with learning words or anything that requires memorization. Content for both sides of the flash card are populated by an input file. The flashcard will be displayed for 3 seconds before showing a new one on the screen. Users can click either the tick button to indicate that they've memorized that card or the cross button if they haven't. Flashcards where the user chose the cross button will be recycled while those ticked will be removed from the current card pool.
